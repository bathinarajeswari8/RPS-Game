# Rock Paper Scissors Game

## Project Overview
This is a simple Python-based Rock Paper Scissors game where the player competes against the computer. The computer's moves are generated using Python's `random` module, making each round unpredictable and engaging.

## Features
- Player vs Computer gameplay
- Random computer choices
- 5 rounds conducted automatically
- Score tracking throughout the game
- Final winner announcement
- Displays the winning margin at the end of the game

## Technologies Used
- Python
- Random Module

## How It Works
1. The player chooses Rock, Paper, or Scissors.
2. The computer randomly selects its choice.
3. The winner of each round is determined according to the game rules.
4. Scores are updated after every round.
5. After 5 rounds, the final scores are displayed along with the overall winner and the score difference.
